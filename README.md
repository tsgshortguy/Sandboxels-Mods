This is the new place to submit mods for [Sandboxels](https://sandboxels.R74n.com), the free in-browser falling-sand simulator. Mods submitted here will be easily usable by players.

For more informatiom on modding, see the [Modding Tutorial](https://sandboxels.wiki.gg/wiki/Modding). For more info about submitting your mod, see [Putting It Online](https://sandboxels.wiki.gg/wiki/Modding/Putting_it_online).

Please read the [Contribution Guidelines](https://github.com/R74nCom/sandboxels/tree/main/.github/CONTRIBUTING.md) before submitting your mod.
